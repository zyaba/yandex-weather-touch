## Ecosystem
Install [Bower](http://bower.io/) and [Gulp](http://gulpjs.com/)
```bash
npm install -g bower
npm install -g grunt
```
Install all of the dependencies
```bash
bower install
npm install
```
Run the default Gulp task to prepare required files
```bash
gulp
```
Open index.html :)
